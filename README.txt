fallUP is a simple game written with the Greenfoot API. It was created as a final project for a university class.

Set Up
To play, open the project.greenfoot file using Greenfoot or simply open the fallUp.jar file.

Instructions
1. Press space to change the “gravity” on the ball.
2. Click on buttons to navigate through the menus.
